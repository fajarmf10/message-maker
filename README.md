# message-maker

## Run group invitation:

### Windows:
```bash
cd src
set MULTI=true
node index.js
```

### Unix:
```bash
cd src
MULTI=true node index.js
```

## Run individual invitation:

### Windows:
```bash
cd src
set MULTI=false
node index.js
```

### Unix:
```bash
cd src
MULTI=false node index.js
```
